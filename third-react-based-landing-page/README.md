# Ship-Ready Apps Landing Page

A bold, Neo-Brutalist landing page for a web app development agency built with React, Vite, Tailwind CSS, and Framer Motion.

## Features

- **Neo-Brutalist Design**: High contrast, thick borders, hard shadows
- **Springy Animations**: Tactile, bouncy interactions using Framer Motion
- **Modular Components**: Clean, maintainable component structure
- **Responsive**: Works on all screen sizes

## Tech Stack

- React 18
- Vite
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide React Icons

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Design System

- **Base Color**: `bg-yellow-50`
- **Borders**: `border-2 border-black`
- **Shadows**: `shadow-[4px_4px_0px_0px_rgba(0,0,0,1)]`
- **Typography**: Space Grotesk (headings), Courier New (monospace)
- **Accent Colors**: Purple, Green, Hot Pink
